#01 - HTML & CSS

Objetivo: Criar uma página de perfil pessoal usando HTML e CSS. A página deve conter as seguintes seções:

  • Cabeçalho: Com o título "Meu Perfil".
  • Foto de Perfil: Uma imagem representando você.
  • Sobre Mim: Uma breve descrição sobre você.
  • Informações de Contato: Links para suas redes sociais e email.
  • Hobbies e Interesses: Uma lista dos seus hobbies e interesses.

#02 - HTML, CSS & JavaScript

Objetivo: Criar uma página de to-do list usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Minha To-Do List".
  • Entrada de Tarefas: Um campo de entrada de texto para adicionar novas tarefas.
  • Botão de Adicionar: Um botão para adicionar a tarefa à lista.
  • Lista de Tarefas: Uma lista que exibe as tarefas adicionadas.
  • Botão de Concluir: Um botão ao lado de cada tarefa para marcá-la como concluída.
  • Botão de Remover: Um botão ao lado de cada tarefa para removê-la da lista.

#03 - HTML, CSS & JavaScript

Objetivo: Criar uma página de calculadora simples usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Calculadora".
  • Display: Um campo que exibe os números e resultados.
  • Teclado Numérico: Botões para os números de 0 a 9.
  • Operadores: Botões para operações de adição, subtração, multiplicação e divisão.
  • Botão de Limpar: Um botão para limpar o display.
  • Botão de Igual: Um botão para calcular e exibir o resultado.

#04 - HTML, CSS & JavaScript

Objetivo: Criar uma página de conversor de moedas usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Conversor de Moedas".
  • Entrada de Valor: Um campo de entrada de texto para o valor a ser convertido.
  • Seleção de Moeda: Dropdowns para selecionar as moedas de origem e destino.
  • Botão de Converter: Um botão para realizar a conversão.
  • Exibição de Resultado: Um campo que exibe o valor convertido.
  • Utilização de API: A conversão deve ser feita usando uma API de taxas de câmbio.

#05 - HTML, CSS & JavaScript

Objetivo: Criar uma página de quiz usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Quiz de Conhecimentos Gerais".
  • Perguntas: Exibir uma pergunta por vez.
  • Opções de Resposta: Botões para selecionar a resposta correta.
  • Feedback: Mensagem indicando se a resposta está correta ou incorreta.
  • Pontuação: Exibir a pontuação atual do usuário.
  • Navegação: Botões para avançar para a próxima pergunta e voltar para a anterior.

#06 - HTML, CSS & JavaScript

Objetivo: Criar uma página de galeria de imagens usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Galeria de Imagens".
  • Exibição de Imagens: Uma grade de imagens com visualização em miniatura.
  • Visualização Ampliada: Ao clicar em uma imagem, exibi-la em tamanho maior.
  • Navegação: Botões para navegar entre as imagens na visualização ampliada.
  • Legendas: Adicionar legendas às imagens.

#07 - HTML, CSS & JavaScript

Objetivo: Criar uma página de relógio digital usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Relógio Digital".
  • Exibição de Hora: Um display digital mostrando a hora atual.
  • Formato de 24/12 horas: Opção para alternar entre formato de 24 horas e 12 horas.
  • Data Atual: Exibir a data atual abaixo do relógio.
  • Estilo Personalizado: Aplicar estilos CSS para personalizar a aparência do relógio.

#08 - HTML, CSS & JavaScript

Objetivo: Criar uma página de blog simples usando HTML, CSS e JavaScript. A página deve conter as seguintes funcionalidades:

  • Cabeçalho: Com o título "Meu Blog".
  • Postagens: Exibir uma lista de postagens do blog.
  • Formulário de Comentários: Um formulário para adicionar novos comentários.
  • Exibição de Comentários: Listar os comentários abaixo de cada postagem.
  • Data das Postagens: Exibir a data em que cada postagem foi publicada.
  • Categorias: Filtrar postagens por categorias.
