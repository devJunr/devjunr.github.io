# WEBSITE PESSOAL
A tela do site de Adriano Jr é apresentada com um estilo retrô que lembra a estética de terminal de computador dos anos 90. O fundo é predominantemente preto, com texto em verde e roxo, evocando uma atmosfera cyberpunk. Aqui está uma descrição detalhada dos elementos presentes na tela:

### Descrição da Tela

**Título Principal:**
- **Adriano Jr:** O nome do desenvolvedor está exibido em uma fonte de estilo pixel art, com a cor verde neon, destacando-se no centro superior da página.

**Opção de Idioma:**
- **"Go to English version":** Um link em vermelho destacado em uma caixa roxa, localizado no canto superior esquerdo, oferecendo a opção de mudar o idioma do site para inglês.

**Seções Principais:**

1. **Sobre Mim:**
   - **Descrição:** A seção "Sobre Mim" está em roxo, com um texto verde que diz: "Estou cursando análise e desenvolvimento de sistemas, e tentando aprender como autodidata na construção de sites e aplicações web. Exploro tecnologias como Python, APIs, SSH, Java, Linux."

2. **Tecnologias:**
   - **Tecnologias mais usadas:** Título em roxo, seguido de uma lista de tecnologias em verde:
     - HTML5
     - CSS3
     - Python
     - Linux
     - Debian
     - Java

3. **Ferramentas:**
   - **VSCode**
   - **GIT**
