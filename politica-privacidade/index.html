Compromisso com a Privacidade Coleta de Dados
A nossa privacidade é importante, principalmente na internet. Esta política descreve como tratarei os dados que você nos envia. Nenhuma coleta de dados é efetuada. No entanto, se você enviar algum dado para nós, ou se nós enviarmos dados a você, esses dados serão tratados com segurança.
Tratamento de Dados Compartilhamento de Dados Alterações na Política de Privacidade
Todos os dados enviados serão armazenados de forma segura e serão excluídos após um período mínimo de 1 dia. Nenhum dado será enviado ou tratado por terceiros. A sua informação é mantida em sigilo e não será compartilhada. Reservamo-nos o direito de alterar esta política a qualquer momento. Recomendamos que você a revise periodicamente para estar ciente de quaisquer mudanças.
